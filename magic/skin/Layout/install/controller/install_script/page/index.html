♠ 페이지 수정
현재 화면을 수정하려면 [관리자]로 로그인 후, 화면 우측 상단의 [페이지수정] 버튼을 클릭하세요.
위지윅에디터에서 입력한 내용 그대로 페이지를 표시할 수 있습니다.
 
♣ 메뉴 수정
상단과 사이드메뉴를 수정하려면 맨위의 [관리자]를 클릭해 관리자페이지로 들어가세요.
관리자페이지의 [메뉴관리]에서 메뉴를 추가,수정할 수 있습니다.
 
◆ 스킨 바꾸기
웹모나의 스킨자료실에서 마음에 드는 스킨을 다운받아 그냥 덮어 쓰세요.
단지 복사하는 것만으로 디자인을 변경할 수 있습니다.
